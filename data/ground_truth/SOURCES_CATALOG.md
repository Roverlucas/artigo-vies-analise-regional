# Ground Truth Sources — Catalog by Country

**Purpose:** documented, verifiable, replicable sources for ground truth construction per country and domain.
**Method:** at least 2 independent official sources per prompt when available. All URLs recorded with access date and SHA-256 of the document version used.
**Domains:** D1 = Public policy & institutions | D2 = Socioeconomic reality | D3 = Environmental context.

## Data collection protocol

**For each country-domain-prompt triple:**

1. **Primary source query:** access the relevant authoritative source for the specific question in the prompt.
2. **Secondary corroboration:** confirm via a second independent source (cross-country harmonized database OR secondary national source).
3. **Disagreement protocol:** if the two sources disagree by >5% or categorically, flag for panel adjudication and prioritize national-level source as primary with secondary as sensitivity.
4. **Version recording:** store document version, access date in ISO 8601, and SHA-256 of the retrieved file in `data/_provenance.json`.
5. **Pre-cutoff subset:** for each prompt, identify whether the answer is available in documents dated before each LLM's training cutoff (contamination control).

**Failure cascade:** if primary source unavailable → secondary → cross-country database → flag prompt as "ground-truth unavailable" and reassign task to open-ended (T2/T3/T4) or exclude.

## License and reuse

All sources listed are either public-domain or openly licensed (CC-BY, CC-BY-SA, or equivalent government open-data license). Data compiled for this benchmark will be redistributed under **CC-BY-4.0** in the Zenodo deposit, with full attribution to primary sources. No paywalled, subscription-based, or proprietary data is used.

**Ethical notes:**
- No personal data extracted.
- No aggregation below administrative unit that could enable re-identification.
- Respect of each source's `robots.txt` and rate-limiting guidelines.
- All APIs accessed via official endpoints with compliant user agents.
