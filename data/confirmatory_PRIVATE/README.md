# Confirmatory Prompts — PRIVATE storage

Per pre-registration §3.6 #4: confirmatory prompts are NOT released
in the public repository before confirmatory data collection completes,
to prevent inadvertent inclusion in any model's training corpus during
the collection window.

This directory is gitignored. Prompts file (`prompts_confirmatory.jsonl`)
exists locally only and is excluded from git via `.gitignore`.

Public release accompanies confirmatory data deposit at Zenodo (CC-BY-4.0).
